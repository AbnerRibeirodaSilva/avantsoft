Backend
Na raiz do backend, execute os seguintes comandos:

npm install  
npx prisma generate  
npx prisma migrate dev  
npm run test  
npm run dev

O servidor estará disponível em: http://localhost:3333

Frontend
O frontend foi desenvolvido em flutter na versão 3.29.3 e testado em um emulador Android. Por isso, a base URL utilizada é:
http://10.0.2.2:3333

Caso seja executado em um dispositivo físico ou emulador iOS, é necessário ajustar a URL para apontar corretamente para o backend (localhost pode variar).

Recomendo fortemente que os testes sejam feitos em um emulador Android para evitar qualquer problema de conexão.
